# KALAVAI Paper Experiments

Reproducible experiment codebase for:  
**"Frozen Foundations: Guaranteed Fusibility for Cooperative LLM Training via Shared Early Layers"**

## Quick Start

```bash
# 1. Setup environment
pip install -r requirements.txt
wandb login  # for experiment tracking

# 2. Run all synthetic experiments (Table 1 in paper)
bash scripts/run_synthetic.sh

# 3. Run all real-model experiments (Table 2 in paper)
bash scripts/run_real.sh

# 4. Run evaluations / benchmarks (Table 3 in paper)
bash scripts/run_eval.sh

# 5. Generate all figures
python scripts/generate_figures.py
```

## Reproducing Results

Every experiment runs with 3 random seeds (42, 137, 2026) and reports mean ± std.
All configs are deterministic given the seed. Hardware: single NVIDIA RTX 5090.

## Environment

- Python 3.11+
- PyTorch 2.x (nightly for RTX 5090 / sm_120 support)
- Hugging Face Transformers, PEFT, Datasets, Accelerate
- lm-eval (EleutherAI Language Model Evaluation Harness)
- Weights & Biases for tracking
